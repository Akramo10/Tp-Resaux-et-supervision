Statut :

Test : Reset arret
Action : Cliquer Reset arret ou choisir 3 dans le terminal.

Statut :

Test : Modification debit valide
Action : Saisir 100, 0 puis 500 dans le terminal, ou utiliser +/- dans le GUI.

Statut :

Statut :
